# 无线访问点

<BlockImage id="wireless_access_point" p:state="has_channel" scale="8" />

允许通过 <ItemLink id="wireless_terminal" /> 无线访问网络。其工作范围与能量消耗由安装的 <ItemLink id="wireless_booster" /> 数量决定。

单个网络可放置任意数量的无线访问点，每个访问点也可安装任意数量的 <ItemLink id="wireless_booster" />，可据此平衡覆盖范围与能耗。

需占用1个[频道](../ae2-mechanics/channels.md)。

也用于绑定[无线终端](wireless_terminals.md)。

# 无线信号增幅器

<ItemImage id="wireless_booster" scale="2" />

用于增加无线访问点的工作范围。
